# Backend Requirements

Last updated: November 13, 2025

# The backend requires the latest Python version and the following dependencies:

## Environment

### Python: Latest stable release

### CUDA: Required for GPU acceleration (ensure it matches your PyTorch version)
```
Python Packages
Flask
torch
torchvision
matplotlib
plotly
numpy
pandas
```

## Installation

*You can install all dependencies directly using the following command in the backend directory:*

```
pip install -r requirements.txt
```
